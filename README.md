# SafraCerta.ai 🚜🌾

O SafraCerta é um assistente inteligente de gestão agrícola projetado para o pequeno e médio produtor rural.

## 🚀 Funcionalidades

- **Registro Inteligente por Voz/Texto**: Use linguagem natural para registrar atividades ("Gastei 200 reais com diesel no talhão 1").
- **MIP (Monitoramento Integrado de Pragas)**: Identificação de pragas via foto com IA local (TensorFlow.js).
- **Dashboard Financeiro**: Acompanhamento de lucro, custos e receitas em tempo real.
- **Gestão de Talhões e Maquinário**: Controle completo da frota e das áreas de cultivo.
- **Modo Offline**: Funciona sem internet e sincroniza automaticamente quando houver conexão.

## 🛠️ Tecnologias

- **Frontend**: Vanilla JS, HTML5, CSS3.
- **Backend**: Node.js, Express.
- **IA**: TensorFlow.js (Vision), Rule-based NLP (Text).
- **Banco de Dados**: LowDB (Local JSON) / Suporte para Firebase.

## 📦 Como Rodar

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor:
   ```bash
   npm start
   ```
3. Acesse `http://localhost:3000` no seu navegador.
